# NLC TPS-II AI Agent — Setup Guide
# Mohanraj · CSE Division

==================================================
STEP 1: Install Python Backend
==================================================

cd backend
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000

API runs at: http://localhost:8000
API docs at: http://localhost:8000/docs   ← Auto-generated by FastAPI!

==================================================
STEP 2: Install & Start Local AI (Ollama)
==================================================

# Download Ollama from: https://ollama.com
# Then run:
ollama pull llama3          # Download AI model (~4GB, one time)
ollama serve                # Start AI server on port 11434

# Test:
curl http://localhost:11434/api/generate -d '{"model":"llama3","prompt":"Hello"}'

NOTE: All AI runs LOCALLY. No internet needed. No data leaves NLC network.

==================================================
STEP 3: Install React Frontend
==================================================

cd frontend
npx create-react-app . --template minimal   # First time only
cp src/App.jsx src/App.jsx                  # Your file is already here
npm install
npm start

Frontend runs at: http://localhost:3000
